# Airbnb NYC Investment Analysis

## Project Overview
This project analyzes Airbnb listings in New York City to identify profitable investment opportunities using data analytics techniques.

## Objective
- Identify high-performing neighborhoods
- Analyze pricing trends
- Evaluate occupancy and revenue potential
- Provide investment recommendations

## Tools Used
- Python
- SQL
- Power BI
- Tableau
- Excel
